浏览器如何计算优先级：千位： 如果声明在 style 的属性（内联样式）则该位得一分。这样的声明没有选择器，所以它得分总是1000。百位： 选择器中包含ID选择器则该位得一分。十位： 选择器中包含类选择器、属性选择器或者伪类则该位得一分。个位：选择器中包含元素、伪元素选择器则该位得一分。

用户行为伪类，有时叫做动态伪类，表现得就像是一个类在用户和元素交互的时候加到了元素上一样。:hover :focus
伪类是一个选择器，它选择处于特定状态的元素，例如它们是它们类型的第一个元素，或者它们被鼠标指针悬停在上面。它们往往表现得好像您在文档的某些部分应用了一个类，通常帮助您减少标记中多余的类，并为您提供更灵活、可维护的代码。

伪元素以类似方式表现，不过表现得是像你往标记文本中加入全新的HTML元素一样，而不是向现有的元素上应用类。伪元素开头为双冒号:: first-line first-letter after before grammer-error selection spelling-error 生成带有::before和::after的内容);

横向书写模式下，映射到width的属性被称作内联尺寸(inline-size)——内联维度的尺寸。而映射height的属性被称为块级尺寸(block-size)，这是块级维度的尺寸。

弹性盒子(Flexbox): 父元素上应用 display: flex
要使用 flexbox，请将 display: flex 应用于要布局的元素的父元素；然后它的所有直接子项都成为弹性项目。
main start and main end; cross start and cross end
flex-wrap: wrap; flex: 200px;
flex-direction: row; + flex-wrap: wrap; = 缩写 flex-flow: row wrap;
flex-direction:row column row-reverse column-reverse; flex-wrap:wrap; flex:20px; 缩写flex-flow: row warp; flex: 1 200px; flex: 1 auto;
flex是一个可以指定最多三个不同值的缩写属性：flex-grow(无单位比例) flex-shrink(无单位比例) flex-basis(最小值)

align-items控制flex项在交叉轴上的位置。stretch center flex-start flex-end
通过将 align-self 属性应用于它们来覆盖单个弹性项目的 align-items 行为：align-self: flex-end;
justify-content 控制 flex 项在主轴上的位置。 flex-start flex-end center space-around space-between
flex项排序: order:1 order值大的flex项比order值小的在显示顺序中更靠后。默认为0，可以为负数，排在0前面
