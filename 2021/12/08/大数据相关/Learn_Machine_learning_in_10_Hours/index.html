01 What is Machine Learning?

Machine Learning is a subset oa artificial intelligence. It focuses mainly on the designing of systems, thereby allowing them to learn and make predictions based on some experience which is data in case of machines.

02 AI vs ML vs Deep Learning

Artificial Intelligence: A technique which enables machines to mimic human behaviour
Machine Learning: Subset of AI technique which use statistical methods to enable machines to improve with experience
Deep Learning: Subset of ML which make the computation of multi-layer neural network feasible
Machine Learning vs Deep Learning: Data Dependency; Hardware Dependency; Feature Engineering; Problem Solving Approach; Execute Time; Interpretability
Types of Machine Learning
Supervised Learning
Unsupervised Learning
Reinforcement Learning
