数量关系

代入排除法：

年龄 余数 多位数 不定方程 不会做的

数字特性：

求偶特性：知差求和 知知求差 2倍类 平均分 不定方程 和差同性
整除特性：一个数能被2(5)整除，末一位能被2(5)整除； 一个数能被4(25)整除，末一位能被4(25)整除；一个数能被8(125)整除，末一位能被8(125)整除；一个数能被3(9)整除，各位数字之和能被3(9)整除；
倍数特性：普通倍数；因子倍数；比例倍数；倍数 百分数 分数 比例 分组 a一定是m的倍数 b一定是n的倍数 a+b一定是m+n的倍数 a-b一定是m-n的倍数

方程法：

普通方程：通用题型 和差倍比 鸡兔同笼 盈亏问题 经济利益 行程问题 优先设置所求量 设小不设大 设中间变量
不定方程(组):限制性不定方程 奇偶性 因子倍数 尾数法 代入法

工程问题：

工程总量 = 效率 * 时间
给定时间型：列表 赋值 总量为时间公倍数
效率制约性：工作效率比 提高20% 42台收割机

行程问题：

路程 = 速度 * 时间
流水行船
顺流速度 = 静水船速 + 水速
逆流速度 = 静水船速 - 水速

相遇追及问题：

相遇距离 = (大速度 + 小速度) * 相遇时间
追及距离 = (大速度 - 小速度) * 追及时间
环形线n次相遇，共同行走的距离 = n * 环线长度
环形线n次追及，追及的距离 = n * 环线长度
直线两端相遇问题：
直线型两端出发n次相遇，共同行走距离 = (2n - 1) * 两地初始距离
等距离求平均速度：
上下坡 往返

排列组合：

捆绑法-相邻问题
插空法-不相邻问题，先对其他元素排列，然后将不相邻元素插入空中
插板法-N个物品分给M个人，没人至少分得一个，N个物品中间有(N-1)个空，在空中插入(M-1)个板，共有C(M-1)(N-1)中情况
错位排列：有N封信和N个信封，每封信都不装在自己对应的信封里，可能的方案数记作D_n, D_2 = 1, D_3 = 2, D_4 = 9, D_5 = 44

概率问题：

某种情况发生的概率 = 满足条件的情况数/总的情况数 = P
分步乘法型：分步概率 = 满足条件的每个步骤概率之积
分类加法型：总体概率 = 满足条件的各种情况概率之和
逆向计算型：1 - 该事件不发生的概率
###经济利润问题：
利润折扣问题
总成本 = 单个成本 * 进口量
总售价 = 单价 * 销售量
利润 = 售价 - 成本
总利润 = 总售价 - 总成本
利润率 = 利润/成本 = (售价 - 成本)/成本 = 售价/成本 - 1
分段计费问题：
找到分段节点，分区间讨论计算

数量关系蒙题技巧：
三奇一偶选其偶，三偶一奇选其奇
最大最小不必看，答案多为中间项
选项中如果有明显的整百整千的数字，代入验证
题目中存在比例关系，选择满足该比例中数字的整除特性的选项
对于复杂的计算问题，答案中尾数不相同，用尾数法
极值问题中，问最小，答案多为第二小的选项；问最大，答案多为第二大的选项。

判断推理

判断推理：图形推理 类比推理 定义判断 逻辑推理
位置规律 样式规律 属性规律 数量规律 特殊规律 空间重构 立体图形
平移 方向 步数 元素组成相同，优先看位置 旋转 翻转 角度 方向
相加 相减 求异 求同

位置规律

元素组成相同：

考点一：平移

  • 方向：上下，左右，顺逆时针、斜对角；步数：恒定、递增、递减

注意1：隐藏元素（空白格子）

注意2：隐藏轨迹（十六宫格或者25宫格，就有内圈和外圈了）

考点二：旋转、翻转
左右翻转的竖轴对称（上下不变左右变）
上下翻转的横轴对称

样式规律

元素组成相似(线条重复)

遍历：补形状；补颜色；

加减同异

“黑白”运算：特征：黑块数量不同；相同位置运算；多部分相同颜色叠加）

属性规律

考点一：对称性

轴对称（等腰图形、箭头）
中心对称（SNZ、平行四边形）
轴加中心对称图形（特殊）
对称轴的细化（对称轴的方向和数量、对称轴与图形关系如线条重合）
考点二：曲直性（属性对称优先，全直全曲）

考点三：开闭性（完整的图形有开口，可以考虑开闭性，如C；全封闭&全开放）

数量规律

数量类识别特征：组成不同不相似，属性没有规律，数量规律明显。

考点：点、线、面、素、角。

考点一：点数量（线与线接触的位置，交点、切点），端点和交点有区别

（1）线条交叉明显

（2）乱糟糟一团线

（3）相切较多

细化考法1：曲直交点（整体点数无规律，且存在曲直相交）

细化考法2：内外交点（有明显外框，框内的点&框上的点&框外的点）

考点二：线数量

直线数特征：多边形、单一直线

特殊考点一笔画（线条之间联通；奇点数0或2，奇点：发射奇数条线的点，端点都是奇点）
特殊考点多笔画：笔画数等于奇点数除以2（奇点数一定是偶数个）
笔画数常见特征：五角星、日、田、圆相切相交、多端点图形

考点三：面数量（封闭空间）

什么时候数面：图形被分割、封闭面明显；生活化图形、粗线条图形中空白区域

细化考法：如果数面无规律，那么

1.面的形状：三角形、四边形

2.面的大小：最大面、最小面（形状、属性、与外框的关系）

考点四：素数量

考点1：元素个数、种类（特征：多个独立的小图形）

考点2：部分数（连在一起叫做一部分，特征：生活化图形、黑色粗线条图形）

考点五：角数量（直线和直线相交才是角；小于180度才是角）

特征：扇形；折线多；改造图（如构图有缺口）

整体数角，然后细化成直角钝角锐角

特殊规律

考点一：图形间关系：相交、相离

相交：相交于点（一线牵图形）、相交于边（数量、长短、整体边还是部分边）、相交于面（面的形状、面的大小、相交面和边框的关系比如保持一致）

考点二：功能元素（黑点白点、箭头、小图形）

标记位置：相交内的吗还是相交外的。

空间重构

想象力：折叠方向固定
靠技巧：排除错误选项
考点1：相对面（两个相对面能且只能看到一个面）

如何判断：同行或同列相隔一个面 or Z字形的两端（紧邻中线）

考点2：相邻面——公共边

平面图中直接相邻的两个面的公共边
平面图中构成直角的两个边是同一条边
一排4个面，首尾会相接
注：折叠后相邻关系不变
考点2：相邻面——公共点

注：公共点引出的线，保持不变

考点2：相邻面——画边法

找个特殊面的唯一点（框上找、独一无二的）
顺时针方向画边，并标出序号
题干与选项对应：面不一致排除

考点3：三视图

观察到的三视图都是平面图
原图没线就没线
被遮挡部分是虚线或空白表示

考点4：截面图

解题原则：正方体、圆柱圆锥圆台

六面体：矩形 梯形 三角形

圆柱：正圆、椭圆、矩阵

圆锥：正圆、椭圆、三角形

圆台：圆、椭圆、梯形

注：斜着切圆柱没有梯形

考点5：立体拼合

判定：给出立体图，需要找到拼合的小图形

凹凸有致的解题原则：互补

难题：俄罗斯方块拼合

看选项，若个数不一样，数个数
假如个数一样，则层层剖析：先最大块

黑白运算：黑块数量相同，优先平移；黑块数量不同：优先黑白运算(相同位置运算)；
属性规律：元素组成不相同 不相似 优先考虑属性 对称性(轴对称 方向和数量 是否自带对称轴 中心对称) 曲直性(全曲线 全直线 半直半曲 外直内曲) 开闭性(全封闭 全开放 完整的图形留了小开口)
数量规律：元素组成不相同、不相似 数量规律明显
点(直线和直线交点 直线和曲线交点 曲线和曲线交点 线条交叉明显 乱糟糟一团线交叉 相切较多 )
线(多边形 但一直线 曲线图形 全曲线图 圆 弧) 面 素 角
