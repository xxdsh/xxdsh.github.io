923. 3Sum With Multiplicity(三数之和的多种可能)

方法一：Math / Combination

Time complexity: O(n + |target|^2)
Space complexity: O(|target|)
step 1: count element frequency
step 2: loop over all possible i, j. k = target - i - j.
condition 1: i <= j <= k
condition 2: k >= 0 && k <= max(A)
case 1: i == j == k , C(count(i), 3)
case 2: i == j != k， C(count(i), 2) * count(k)
i != j == k， C(count(j), 2) * count(i)
case 3: i != j != k, count(i) * count(j) * count(k)

1395. Count Number of Teams(统计作战单位数)

Solution 1: Brute Force

Solution 2: Math

For each soldier j, count how many soldiers on his left has smaller ratings as left[j], count how many soldiers his right side has larger ratings as right[j]

ans = sum(left[j] * right[j] + (j – left[j]) * (n – j – 1 - right[j])

Time complexity: O(n^2)
Space complexity: O(1)

1467. Probability of a Two Boxes Having The Same Number of Distinct Balls(两个盒子中球的颜色数相同的概率)

方法一：Permutation(TLE)

construct a list of balls and generate all permutations. Put first n/2 balls into box1, put rest of the balls into box2 and verity the result.
Enumerate all permutations of the balls, count valid ones and divide that by the total.
Time complexity: O((86)!) = O(48!)
After deduplication: O(48!/(6!)^8) ~ 1.7e38
Space complexity: O(86)

方法二：Partitioning

Once we decide the color and the numbers of balls in each box we can determine whether it is valid or not without considering the orders. Which means we can do permulation later, or just count #of possible permutations without actually doing it.

For each color, put n_i balls into box1, the left t_i – n_i balls go to box2.
permutations = fact(n//2) / PROD(fact(n_i)) * fact(n//2) * PROD(fact(t_i – n_i))
E.g
balls = [1×2, 2×6, 3×4]
One possible combination:
box1: 1 22 333
box2: 1 2222 3
permutations = 6! / (1! * 2! * 3!) * 6! / (1! * 4! * 1!) = 1800

Time complexity: O((t+1)^k) = O(7^8)
Space complexity: O(k + (t*k)) = O(8 + 48)
