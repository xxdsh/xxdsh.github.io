方法一：Sliding Window

  1. Transfer the coordinate system, treat location[0], location[1] as original
  2. Compute the angle of each point w.r.t to new origin,
  3. sort all the angles
  4. Maintain a window [l, r] s.t. angle[r] - angle[l] <= FOV
