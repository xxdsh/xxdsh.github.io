计算机视觉

图像分类、物体检测、语义分割、文字识别

物体检测(object detection)

解决实例分割(instance segmentation)、场景理解(scene understanding)、目标跟踪(object tacking)、图形标注(image captioning)等问题的基础

检测输入图像中是否存在给定类别的物体，如果存在，则输出物体在图像中的位置信息

位置信息：矩形边界框(bounding box)的坐标轴
单步模型(one-stage)和两步模型(two-stage)

  • 单步模型

    • 没有独立地、显式地提取候选区域(region proposal),直接由输入图像得到其中存在的物体的类别和位置信息的模型
    • OverFeat/SSD(single shot multibox-detector)/YOLO(You Only Look Once)/CornerNet/RefineDet/ExtremeNet
    • 计算效率

  • 两步模型

    • 有独立地、显式地提取候选区域过程，即先在输入图像上筛选出一些可能存在物体的候选区域，然后计算每个候选区域，判断其是否存在物体，如果存在，就给出物体的类别的位置修正信息。
    • R-CNN/SPPNet/Fast R-CNN/Faster R-CNN/R-FCN/Mask R-CNN/Faster/PANet/Cascade R-CNN/Mask Score R-CNN
    • 检测精度

图像分割

像素级别的图像识别，即标注出图像中每个像素所属的对象类别
前景分割(foreground segmentation)、语义分割(semantic segmentation, 注重类别之间的区分)、实例分割(instance segmentation，注重个体之间的区分)、全景分割(panoptic segmentation，语义分割和实例分割的结合)
编码器-解码器结构：FCN(Fully Convolutional Networks)、U-Net、SegNet
利用快捷连接向解码器中引入编码器提取的特征

  • 将编码器提取的特征进行复制，叠加到之后的卷积层提取出的特征上，作为解码器的输入
  • 最大池化索引(max-pooling index)，快捷连接传递的不是特征本身，而是最大池化时所使用的索引(位置坐标)

DeepLab-Google团队

  • DeepLab v1

    • 空洞卷积(Atrous Covolution)：解决编码过程中信号不断被下采样、细节信息丢失的问题
    • 全连接条件随机场(fully connected CRF)：提高模型捕获局部结构信息的能力
    • 输入图片->深度卷积神经网络->粗分割图->最终的分割图->全连接条件随机场->双线性插值

  • DeepLab v2

    • 空洞空间金字塔池化(Atrous Spatial Pyramid Pooling, ASPP)：解决不同检测目标大小差异的问题

光学字符识别(Optical Character Recognition, OCR)

将包含键入、印刷或场景文本的电子图像转换成机器编码文本的过程：文本检测 + 文本识别

  • 文本检测：浅层次的图像处理和分割方法：基于二值化的连通区域提取、基于最大极值稳定区域(Maximally Stable Extremal Regions, MSER)的字符区域提取；基于候选框、基于像素
    方向梯度直方图Histogram of Oriented Gradient, HOG
  • 文本识别：基于字符分割的；端对端的文本识别：CRNN(CNN+RNN+CTC(Connectionist Temporal Classification))
  • 端对端文本检测+识别算法：FOTS(Fast Oriented Text Spotting):共享卷积层、文本检测分支、感兴趣区域旋转模型以及文本识别分支

图像标注

融合计算机视觉、自然语言处理和机器学习的综合任务
一张图片 ——> 一段描述性文字
评测指标

  • BLEU(Bilingual Evaluation Understudy)
    • N-gram匹配精度的计算；针对标注文本长度小于参考注释的惩罚机制；为了平衡N-gram不同阶之间的精度差别而采用的几何平均
    • 在语料库这一级别上与人工注释具有更好的相关性
  • METROR(Metric for Evaluation of Translation with Explicit ORdering)
    • 精确匹配模块 + porter stem模块 + 基于wordNet的同义词模块
    • 与人工注释在语句层面上具有更好的相关性
  • ROUGE(Recall-Oriented Understudy for Gisting Evaluation)
    • 基于召回率的相似度度量方法
    • 通过比较重叠的N个词中单词序列和单词对来实现
    • 主要考察图像标注的充分性和真实性
  • CIDEr(Consensus-based Image Description Evaluation)
    • TF-IDF 区分对待不同的词
  • SPICE(Semantic Propositional Image Caption Evaluation)
    • 排除N-gram造成的重叠敏感问题，直观，具有可解释性，忽略流畅性
    • 图像标注和参考标注 —–> 语法关系依赖树 ——> 情景图

人体姿态识别

Pose Estimation: 检测图像或视频中人体关键点的位置，构建人体骨架图
动作识别、人机信息交互、异常行为检测

  • 自底向上(基于部件part-based方法)
    • 首先检测出图像或视频中人体的关键点，然后对不同关键点进行匹配，将属于同一个人的关键点连接起来。
    • 关键点回归：DeepPose
    • 关键点检测：PAFs(Part Affinity Fields)/Dense Pose/Associative Embedding
  • 自顶向下
    • 人体检测
    • 关键点检测
