生成式对抗网络

可微生成网络

  • 生成式对抗网络(Generative Adversarial Network, GAN)

    • 优化目标

  • 自编码器(AutoEncoder, AE)

    • 编码器 + 解码器 = 压缩 + 解压
    • 有向图模型，对隐藏空间进行建模
    • 数据去噪；可视化降维
    • 只会模仿，不会创造，无法直接生成任意新样本
    • 优化目标是最小化真实样本与生成样本之间的重构误差

  • 变分自编码器(Variational AutoEncoder, VAE)

    • 生成新样本
    • 有向图模型，对隐藏空间进行建模
    • 编码器的输出不再是隐空间中的向量，而是所属正态分布的均值和标准差
    • 优化目标 = 重构误差 + 隐变量空间的约束目标

JS散度(Jensen-Shannon divergence)
MATHJAX-SSR-294
取值非负，两个分布相等是取0，最小值-2log2
MATHJAX-SSR-295
用上述公式替代生成器的损失函数，具有相同的纳什均衡点，问题：梯度不稳定；惩罚不平衡导致的模式坍塌(缺乏多样性)；不还判断收敛性；难以评价生成数据的质量和多样性

GAN在实际应用中的问题：

  • 理论上的收敛性不能保证实际应用时的收敛性：优化过程是在参数空间而不是概率密度空间
  • 参数空间可能是非凸的；交替的优化过程；博弈过程陷入局部纳什均衡，出现模式坍塌
  • 何时停止，好坏评估，缺乏理想的评价方法和准则

生成式对抗网络的改进

角度：目标函数、模型结构、训练技巧

目标函数

  • 基于f-散度的GAN
    • 用凸函数来度量两个分布与之间的距离
  • 基于积分概率度量(IPM)的GAN
    • WGAN: Wasserstein距离 = 推土机距离
    • McGAN

模型结构

  • DCGAN(Deep Convalutional GAN)
  • Stacked GAN
  • Progressive GAN

训练技巧

  • 特征匹配技术：将原目标函数中对最后一层输出值的比较、改为对中间层输出向量的比较
  • 单边标签平滑
  • 谱归一化：归一化判别器权重，满足Lipschitz连续性

生成式对抗网络的效果评估

  • IS(Inception Score)

    • 没有将真实样本与生成样本比较，仅量化生成样本的质量和多样性

  • FID(Frechet Inception Distance)

    • 将真实样本与生成样本比较，对网络倒数第二层的响应即特征图进行操作

    • 比较真实样本与生成样本的特征图的均值和方差

    • 无法实现对图像细节的评估
      MATHJAX-SSR-297

  • 其他

    • 模式分数
    • 最大均值差异
    • 最近邻双样本检验
    • 切片W-距离

生成式对抗网络的应用

  • 计算机视觉：图像和视频生成；图像与图像或文字之间的翻译；物体检测；语义分割

  • 自然语言处理：文本建模；对话生成；问答系统；机器翻译

难点：无法明确区分图像类别；难以捕捉到图像语义结构；质地和细节不合理

  • SNGAN：谱归一化 + 折页损失

  • SAGAN：谱归一化 + 折页损失 + 自注意力机制(捕获全局的语义和解雇信息)

  • BigGAN：增大GAN的参数规模，增大批尺寸，增加通道数；输入噪声的嵌入和截断技巧；调节各层权重的奇异值的大小；正则化方法

图像到图像翻译方法：

  • 超分辨率重建
  • 图像补全
  • 图像风格迁移

半监督学习方法：低密度分离方法、基于生成模型的方法、基于平滑假设的方法
生成数据的目的在于提升判别器对样本真实分布的学习能力
