自然语言处理(Natural Language Processing)

核心任务：语言模型、语言形态学、语法分析、语义分析
应用：机器翻译、信息检索、问答系统、对话系统

语言的特征表示

词袋模型(Bag-of-Words)
词嵌入(Word Embedding): 出现在相似的上下文中的词含义相似，以此为根据将词从高维稀疏的独热向量映射为地位稠密的连续向量，从而实现对词的语义建模。
基于词出现频次的词嵌入模型
潜在语义分析模型(Latent Semantic Analysis,LSA)
基于词共现矩阵的方法：Glove
基于对上下文中出现的词的预测
Word2Vec - CBOW(Continuous Bag-of-Words)模型 和 Skip-Gram模型
CBOW模型的训练相对Skip-Gram模型更快，Skip-Gram模型对低频词迭代更充分
语言模型(Language Model)

  • 概率分布模型，评估语言中的任意一个字符串的生成概率p(S)
  • 参数空间过大会导致稀疏问题
  • 马尔可夫假设 N元(N-gram)语言模型
  • 可以评估语言的生成概率，直接用于生成符合人类认知的语言
  • 语言模型的训练不依赖额外的监督信息，因此适合用于学习语句的通用语句表达
  • ELMo(Embeddings from Language Model)/GPT(Generative Pre-trained Transformer)/BERT(Bidirectional Encoder Representations from Transformers)
— ELMo GPT BERT
输入层 词序列 词序列 词序列+句子向量+位置向量
网络结构 双向LSTM Transformer单向编码器 Transformer双向编码器
预训练任务 双向语言模型 单向语言模型 掩码语言模型+下一句预测
融合方式 特征融合 参数微调+辅助任务 参数微调

机器翻译

统计机器翻译：翻译模型 + 语言模型 + 调序模型
编码器-解码器结构(Encoder-Decoder)
注意力机制：1）生成上下文向量，为解码提供额外的信息；2）允许任意编码节点到解码节点的直接链接，解决了长距离语境依赖问题
Transformer：多头自注意力机制(multi-head self-attention):1)通过自注意力将句中相隔任意长度的词聚类缩减为常量；2)通过多头结构捕捉到不同子空间的语义信息；3)缺少对位置信息的捕捉能力，显式地对词的不同位置信息进行了编码；4)并行能力
OOV(Out-Of-Vocablary)未登录词：1)损失语义信息2)影响用户体验；解决：
1)将翻译的基本单元从词级别变为字词级别或者字符级别 - BPE(Byte Pair Encoding)算法；2)将源语言中的词拷贝到目标语言

双语语料不足：1)通过爬虫自动挖掘和产生更多的双语语料；2) 构造伪双语平行语料(利用目标语言端的单语语料反向翻译源语言；利用数组增强的方式对原始语料进行改造)；3)在模型层面解决语料不足的问题

问答系统

Question Answering System : 更具用户的问题，从一个知识库或者非结构化的自然语言文档集合中查询并返回答案的计算机软件系统
问题分类 + 段落检索 + 答案抽取
自注意力机制：用较短的信息传递路径来学习文本中的长距离语境依赖
尺度缩放点积注意力(scaled dot-product attention)
QANet-核心：编码器模型-组成：卷积层 + 自注意力层 + 前向全连接层
层级标准化(Layer Normalization)；残差连接；深度可分离卷积(Depthwise Separable Convolutions)
问题和段落协同编码：BiDAF(Bi-Directional Attention Flow);DCN(Dynamix Coattention Network)

对话系统

Dialogue System : 可以通过文本、语音、图像等自然的沟通方式自动地与人类交流的计算机系统
信息领域-开放与闭合；设计目标-任务型与非任务型
对话理解 + 策略学习 + 对话生成
强化学习
重要性采样(Importance Sampling)
经验回放(Experience Replay)
蒙特卡洛方法
ACER(Actor-Critic with Experience Replay)
