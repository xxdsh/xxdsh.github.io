87. Scramble String(扰乱字符串)

isScramble(s1, s2)
if s1 == s2: return true
if sorted(s1) != sroted(s2): return false
We try all possible partitions:

  1. s1[0 : k] v.s s2[0 : k] && s1[k : L] vs s2[k : L] //分割成两半，当前分割点不做swap
  2. s1[0 : k] vs s2[L- k : L] && s1[k : L] vs s2[0 : L- k] //分割成两半，当前分割点做swap

T(n) = 4n * T(n/2) + O(n) ~ O(nlog(4n))

Time complexity: O(n^5)
Space complexity: O(n^4)

1542. Find Longest Awesome Substring(找出最长的超赞子字符串)

方法一：Prefix + Hashtable

Define a state of prefix string input[0-i], update the state when extend the prefix

index[s] = first/last index when state S occureed

Let’s assume state S_i, S_j occurred at [i, j], i < j, condition(s_i, s_j) == True, input[i+1~j] is a solution and its length is j - (i + 1) + 1 = j - i

index[s_init] = -1 # base state “occurred” before the start, this handles the case when entire prefix is a solution.

ans = max/min[i - index[s_cur]] 0 <= i < n

For a palindrome all digits must occurred even times expect one. We can use a 10 bit mask to track the occurrence of each digit for prefix s[0~i]. 0 is even, 1 is odd.

758. Bold Words in String(字符串中的加粗单词)

check whether a char will be bolded or not.

Try all possible substring of S(S的子字符串的长度应当小于等于单词的长度), in words or not

pre-processing: bold = [0,0,0,0,0] -> 需要加粗的index变为1

construct the ans:

Add: ‘<b>‘ to pattern [0,1,……] or [1,……] （first bold char in the sequence）

Add: ‘<b>‘ to pattern [……, 1,0] or [……,1] （last bold char in the sequence）

Time complexity: O(nL^2)

Space complexity: O(n + d)

d: size of dictionary

L: max length of the word which is 10.
